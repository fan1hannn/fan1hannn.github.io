    科目一速通

    一、扣分

    1分

    • 不能做
      1.不系 安全带
      2.不戴 头盔
      3.不按规定 会车
      4.不按规定 使用灯光
    • 违反
      1.违反 禁令标志、禁止标线
      2.普通路 违规倒车调头
      3.普通车没安检
    • 超过
      1.特殊车-普通路 超速 10%-20%

    3分

    • 不能做
    1. 特殊路不按规定车道行驶
    2. 普通路逆行
    3. 不按规定超车、让行；借道超车
    4. 不按规定避让校车
    5. 不按规定装车牌
    6. 事故后，不按规定设置警告
    7. 不礼让行人
    • 超过
    1. 普通车 超载 20%-50%
    2. 货车 超载 30%-50%
    3. 普通车-普通路 超速 20%-50%
    • 其他
    1. 货车 4h-20min
    2. 高速慢行
    3. 打电话
    4. 特殊车没安检

    6分

    • 不能做
    1. 运危险品没挂警示标志
    2. 闯红灯
    3. 驾驶证被扣开车
    4. 违法占用应急车道
    5. 轻微伤或者财产损失 不构成犯罪
    • 超过
    1. 校车、公路/旅游客车 超载 <20%
    2. 7座以上 超载 20%-50%
    3. 货车 超载 50%
    4. 特殊车-普通路 超速 20%-50%
    5. 特殊车-特殊路 超速 <20%

    9分

    • 不能做
    1. 特殊路停车
    2. 未挂/遮挡车牌
    3. 准驾不符
    • 超过
    1. 7座以上 超载 50%-100%
    2. 特殊车-普通路 超速 >50%
    • 其他
    1. 准驾不符
    2. 无证开校车
    3. 客车 危险品车 4h-20min

    12分

    • 不能做
    1. 酒后开车
    2. 伪造证件
    3. 特殊路掉头
    4. 代替处罚
    • 超过
    1. 校车、公路/旅游客车 超载 >20%
    2. 特殊车-特殊路 超速 >20%
    3. 普通车-特殊路 超速 >50%

    计分

    计分周期：一年

    满分教育

    扣留驾驶证 →7天学习
    （每多一次+7d，max<60d)
    （现场>2d，网络>3d，自主<2d）

    24分→道路驾驶技能考试
    36分→场地驾驶技能和道路驾驶技能考试

    减分

    一年最多减六分

    • 不可以参加：
    1. 两年内，参加两次以上满分教育
    2. 驾驶证被扣
    3. 实习期
    4. 没年检
    5. 有超期没有办注销的车
    6. 三年内，一次扣十二分（酒、逃、伪造、消分）或者在满分教育中弄虚作假
    • 机制
    1. 网络：三日累计满30分钟，且考试合格的，一次扣减一分
    2. 现场：1h/2分
    3. 公益：1h/1分

    罚款

    0-200

    1. 车身涂鸦、广告遮挡视线
    2. 未按规定办理转让、变更登记
    3. 不及时年检
    4. 撞后，应撤未撤，造成交通堵塞

    20-200

    1. 实习期牵引挂车；独自上高速
    2. 信息变化没有及时换信息
    3. 换证后仍使用原证
    4. 未带驾驶证、行驶证
    5. 未按规定申报信息

    200-500

    1. 驾驶证被扣期间使用隐瞒或者欺骗等手段补领
    2. 身体变化不适合开车，仍然开

    200-2000

    • +15日以下拘留
    1. 造成交通事故，暂不构成犯罪
    2. 驾驶证被扣期间开车
    3. 违反交通管理，强行通行，
    4. 非法截留、扣留机动车，造成交通严重堵塞和财产损失，
    • +吊销驾驶证
    1. 超速50%以上
    2. 把驾驶证给无驾照或被吊销的人
    3. 拼装车或报废车上路行驶
    • 非法安装报警器、标志性灯具；故意损毁、移动、涂改交通设施的

    其他

    1. 虚假材料 500以下（1年不申）

    2. 考试贿赂舞弊，2000以下；组织，3-5倍，10万以下

    3. 消分，3倍以下，5万以下；组织 ，5倍以下，10万以下

    4. 组织替考的，3倍以下，2万以下

    5. 教育中弄虚作假，成绩无效，1000以下

    6. 代替参与教育的，2000以下

    7. 饮酒累犯（饮酒后驾车被罚，再次饮酒驾车）1000-2000罚款，10日以下拘留，并吊销驾驶证

    8. 伪造变造检验、保险标志，1000-3000罚款，10日以下拘留，扣留车

    拓展

    • 扣车:
    1. 无机动车号牌
    2. 无检验合格标志、保险标志
    3. 无行驶证、驾驶证
    4. 使用假证
    • 扣驾驶证:
    1. 酒后驾驶；
    2. 将机动车交由未取得机动车驾驶证或者机动车驾驶证被吊销、暂扣的人驾驶的；
    3. 超速50%
    4. 开拼装或废弃车；
    5. 一年累计12分
